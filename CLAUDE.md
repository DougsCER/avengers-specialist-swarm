# Avengers Assemble — Specialist Swarm

Hackathon project. An Avengers-themed adaptation of the [specialist-swarm](https://github.com/rosscrooke/specialist-swarm) brief. This repo is a fork of `DougsCER/avengers-specialist-swarm` (itself forked from `rosscrooke/specialist-swarm`).

**Working directory:** `/Users/lst/avengers-specialist-swarm/`

## The brief (upstream)

- **Architecture:** coordinator + 3–5 specialist sub-agents + per-specialist Skills.
- **Tech:** Claude Managed Agents (multi-agent, beta header `managed-agents-2026-04-01`), custom Skills API, Anthropic Python SDK.
- **Time budget:** 60-minute hackathon sprint.

## Our twist

Instead of a Deal Desk → `.docx`, we build **"Avengers Assemble: how the heroes change the world"** as an **interactive HTML presentation**. Nick Fury coordinates; heroes each contribute their domain; the UI visualises the fan-out and synthesis.

### Hero → specialist mapping

| Hero | Role | Skill |
|------|------|-------|
| **Nick Fury** | Coordinator — orchestrates, synthesises, does not do the work | — |
| **Iron Man (Tony Stark)** | Engineering / tech specialist | `stark-engineering` |
| **Captain America** | Strategy & ethics specialist | `cap-strategy` |
| **Black Widow** | Intel & recon specialist | `widow-intel` |
| **Bruce Banner / Hulk** | Research / heavy compute specialist | `banner-research` |

Start with 3 (Iron Man, Cap, Black Widow). Add Banner if time allows.

### Output: interactive HTML presentation

Single self-contained `outputs/index.html`:
- Hero cards light up as each specialist thread completes.
- Visualises the parallel fan-out (the events stream IS the demo).
- Ends on the synthesised "how we change the world" brief from Nick Fury.
- Comic-panel layout, Avengers palette, works on a projector. No CDN dependencies.

## Build order

1. `python setup_environment.py` — creates the sandbox environment, writes `.environment_id`
2. `python create_specialists.py` — creates the 4 hero agents, writes `.specialist_ids.json`
3. `python create_coordinator.py` — creates Nick Fury with the heroes in his roster, writes `.coordinator_id`
4. `python upload_skills.py` — uploads `skills/` and attaches each to its hero
5. `python run_mission.py` — runs the swarm, streams events, writes `outputs/index.html`
